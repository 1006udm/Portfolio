Machine Learning

AI in Art History

This project explores the application of Artificial Intelligence (AI) and how can be applied in the feild of art history by analyzing color palettes and stylistic features to predict the period, style, and geographical location of artworks. AI will uncover correlations between visual traits and historical contexts, advancing the understanding of art evolution.

Software Engineering

Client Server Database Interactions

The project involves designing a generic client-server architecture using Java and MySQL for user authentication, data storage, and email generation. The server will manage business logic and database interactions, with a focus on secure login and dynamic email functionality. UML diagrams, including class and use case diagrams, will be used to document the system's structure and flow. This project was developed in collaboration with Justin Bouse, Tang Favish, and Micha Wisniewski.
